# Warehouse Robot System (Prototype)

**PROTOTYPE ONLY** - Created for AI Technical Assessment

## Project Overview
A prototype system for warehouse robotics that combines:
- Computer Vision (object detection)
- Machine Learning (object classification)
- RAG (knowledge retrieval)

## Files Included
- `cv_module/` - Object detection using OpenCV
- `ml_module/` - ResNet50 classifier for fragile/heavy/hazardous
- `RAG_module/` - Knowledge base retrieval system
- `integration_module/` - Complete pipeline
